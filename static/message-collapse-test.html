预期：消息右上角应显示折叠按钮，点击后显示摘要

测试用例 2：AI 回复消息折叠

AI 回复示例：

我来帮你实现斐波那契数列的两种方法：

# 递归方法
def fibonacci_recursive(n):
    if n <= 1:
        return n
    return fibonacci_recursive(n-1) + fibonacci_recursive(n-2)

# 动态规划方法
def fibonacci_dp(n):
    if n <= 1:
        return n
    dp = [0] * (n + 1)
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]

性能比较：递归方法的时间复杂度是 O(2^n)，而动态规划方法的时间复杂度是 O(n)。

预期：长回复消息右上角应显示折叠按钮，点击后显示摘要
